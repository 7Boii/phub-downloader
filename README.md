# PHub Downloader

Open-source browser extension to download videos from PornHub. **100% client-side, no backend, no third-party tools.**

Everything runs locally in your browser — no external servers, no paid APIs, no sketchy executables. The extension simply reads the CDN URLs that PornHub already serves to your browser and lets you save the file. Nothing is uploaded anywhere.

## Features

- Quality selection (720p, 1080p, etc.)
- Download progress with speed indicator
- Multi-language UI (auto-detects site language)
- No login required for free videos
- Zero dependencies, zero external requests

## Install

1. Open `chrome://extensions/`
2. Enable **Developer mode**
3. **Load unpacked** → select this folder
4. Open a video page → click the extension icon → pick a quality → Download
